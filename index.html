Shopping cart does not update quantity when clicking "+" button

Environment: Chrome 120.0, Windows 11, Production (v2.1.3)
Steps to Reproduce:
  1. Open the website and log in with valid credentials
  2. Navigate to any product page
  3. Click "Add to Cart" button
  4. Open the shopping cart
  5. Click the "+" button to increase quantity
Expected Result:

Product quantity increases by 1, total price updates accordingly

Actual Result:

Quantity remains unchanged. Console shows "TypeError: Cannot read property 'quantity' of undefined"

Severity: High Priority: High
2 / 3

Password reset email contains broken link

Environment: Firefox 121.0, macOS Sonoma, Staging (v2.2.0-beta)
Steps to Reproduce:
  1. Go to login page
  2. Click "Forgot Password"
  3. Enter valid email address
  4. Check email inbox
  5. Click the reset password link
Expected Result:

User is redirected to password reset form

Actual Result:

404 Page Not Found error. Link contains duplicate domain: "https://example.com/https://example.com/reset?token=..."

Severity: Critical Priority: High
3 / 3

Search results show deleted products

Environment: Edge 120.0, Windows 10, Production (v2.1.3)
Steps to Reproduce:
  1. Admin: Delete a product from catalog
  2. User: Search for the deleted product by name
  3. Observe search results
Expected Result:

Deleted product does not appear in search results

Actual Result:

Deleted product appears in results. Clicking it leads to 404 error page

Severity: Medium Priority: Medium
